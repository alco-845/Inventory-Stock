# Inventory-Stock
Aplikasi ini bernama **Inventory Stock** yang berfungsi untuk mendata barang masuk, barang keluar, dan history transaksi

Fitur unggulan :
- History transaksi
- Print menggunakan printer bluetooth
- Export excel
- Backup
- Restore
- Reset

Menu yang ada di aplikasi ini :
- Master, dengan sub menu :
  - Identitas toko
  - Kategori
  - Barang
  - Satuan
  - Supplier
  - Pelanggan
- Transaksi, dengan sub menu :
  - Penyimpanan
  - Penjualan
  - Pemasukan
  - Pengeluaran
- Laporan, dengan sub menu :
  - Laporan pelanggan
  - Laporan barang
  - Laporan distributor
  - Laporan penjualan
  - Laporan keuangan
- Utilitas, dengan sub menu :
  - Backup
  - Restore
  - Reset
  - Petunjuk
  - Upgrade ke pro

Aplikasi ini menggunakan :
- Java
- SQLite
